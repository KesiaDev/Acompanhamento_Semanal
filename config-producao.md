## 🌐 **Configurar Acesso de Rede Local:**

### **1. Descobrir seu IP:**
```powershell
ipconfig
```

### **2. Procurar por:**
```
Adaptador Ethernet:
   Endereço IPv4: 192.168.1.XXX
```

### **3. Compartilhar com a Equipe:**
```
http://192.168.1.XXX:3000
```
